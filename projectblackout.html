Project Blackout

Game BI Design
Project Blackout original name is "Point Blank" in Korea. However, there is the contract issue to use same name in US and Canada. Therefore, it changes to "Project Blackout" for US and Canada game users. Also, recreate Game BI to keep not only the original game concept but also different style.

Problems

No more useing the origianl game name and BI becuase of the contract issue. Moreover, they don't have the original logo file to edit for new name "Project Blackout". Additionaly, keeping the original game concept to recreate new logo.

Solution

Keeping the original logo shape "P" "B" and change texture to metal. Also, emphasize main item such as gun and nife on new logo. Moreover, I add the fire and scatching to the behind logo for more attractive.
